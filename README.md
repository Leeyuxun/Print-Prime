# Print Prime

打印一定范围内的素数，理论上可以打印$2^{64}$以内的所有素数到prime文件，但是由于个人电脑的性能问题，只打印到了$2^{33}$

## 1. 设计思路

- 编程语言：C（底层语言，耗时最少）
- 运行系统：Windows 10
- 方法：埃拉托斯特尼筛法（将每个素数的倍数标记为合数，时间和空间利用率远超试除法）
